# 🌍 Global Solution - Shelter Registration System

This project was developed as part of the **Global Solution** course at FIAP. The system aims to facilitate the registration and access to **temporary shelters** and **official shelters**, enabling a faster and more efficient response in emergency situations.

## 📌 Objective

The project provides an accessible platform where **regular users** and **government entities** can:

- Register shelters (official or improvised);
- View a list of nearby shelters;
- Check detailed information about shelters;
- Collaborate by sharing useful information;
- Integrate with government APIs to fetch official data.

---

## 🛠 Features

- 👤 User registration and login;
- 🏠 Register improvised shelters by regular citizens;
- 🏢 Register official shelters by authorized entities;
- 🔍 Search and list shelters;
- 📍 Locate and suggest nearby shelters;
- 🔄 Integration with public APIs for official data;
- 🔐 Access control to protect sensitive information.
